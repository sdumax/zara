# Goals

The planning hierarchy. Everything cascades down — yearly sets the direction, quarterly breaks it
into 90-day sprints, weekly translates that into action, daily executes it.

```
Yearly  →  "Where am I going this year?"
  ↓
Quarterly  →  "What do I need to do in the next 90 days to get there?"
  ↓
Weekly  →  "What specifically am I doing this week?"  (lives in /weekly)
  ↓
Daily  →  "What am I doing today?"  (lives in /goals/daily)
```
