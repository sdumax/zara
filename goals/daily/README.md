# Daily Notes

One file per day. Fast to open, fast to close. The goal is clarity at the start and a clean handoff
to tomorrow at the end.

**Naming:** `YYYY-MM-DD.md` — e.g. `2026-03-13.md`

---

## Daily Note Template

Copy this into today's file, or ask Zara to create it:

```markdown
# [DATE] — [Day of week]

## Today's Top 3
> The 3 things that, if done, make today a win. Pick these first.
1.
2.
3.

## Full Task List
> Everything else — pull from weekly plan + anything that came up
- [ ]
- [ ]

## Meetings / Commitments
| Time | What | Prep needed? |
|------|------|-------------|
| | | |

## Brain State
[ ] Fired up  [ ] Solid  [ ] Slow start  [ ] Running on fumes

## Notes / Captures
> Anything that comes up during the day — ideas, follow-ups, random thoughts

---

## End of Day

**Done:** ✓
**Didn't get to (carry to tomorrow):**
**One thing I learned or noticed today:**
```

---

## How to Start Your Day with Zara

Tell Zara: *"Morning — let's do the daily."*

Zara will:
1. Pull your top items from the current weekly plan
2. Flag anything overdue
3. Help you pick your Top 3
4. Generate today's note file
