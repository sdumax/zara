# Inbox

**The one rule: capture now, sort later.**

---

## How to Use This

When something hits — a task, an idea, a thing you need to remember, a message to send, anything — drop it here. Don't try to organise it in the moment. Just get it out of your head.

Zara processes the inbox with you during weekly reviews or on demand.

---

## Capture Format

Just write it. But if you want structure:

```
[DATE] [TYPE: task/idea/note/follow-up] — [The thing]
e.g.
2024-01-15 [task] — Follow up with client about invoice
2024-01-15 [idea] — Build a CLI tool that summarises git diffs in plain English
2024-01-15 [note] — Interesting agri-tech startup model: subscription soil testing kits
```

---

## Processing Protocol

When reviewing the inbox, each item gets one of:
- **→ area/[folder]** — belongs to a specific area, move it there
- **→ templates/project-brief.md** — idea with legs, flesh it out
- **→ weekly/[week]** — a task for this week
- **→ archive** — processed, no further action needed
- **→ delete** — not worth keeping
