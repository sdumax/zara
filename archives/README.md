# Archives

Completed projects, closed clients, finished content, resolved decisions — anything that's done
but worth keeping for reference.

**Rule:** Nothing gets deleted. Things get archived.

---

## Structure

```
archives/
├── freelance/      ← completed client engagements
├── content/        ← published pieces, completed builds
├── ideas/          ← ideas evaluated and parked (not deleted)
├── weekly/         ← past weekly plans (auto-move after 4 weeks)
└── decisions/      ← resolved decisions
```

---

## How to Archive

When something is done, move it here with a quick header note:

```
# [Original Title]
**Archived:** [DATE]
**Outcome:** [what happened / what was the result]
---
[original content below]
```
