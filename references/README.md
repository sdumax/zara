# References

Static reference material Zara can pull from at any time. Unlike `context/` (which is about you),
references are external — things you look up, standards you follow, info you return to often.

---

## Structure

```
references/
├── README.md           ← you are here
├── tech/               ← language docs, patterns, boilerplate snippets
├── business/           ← contract terms, invoicing, business model references
├── content/            ← platform specs, hooks formulas, editing guides
└── agriculture/        ← farming research, market data, regulations
```

---

## What Goes Here

- Platform specs (e.g. YouTube video length limits, LinkedIn character counts)
- Contract clauses you reuse
- Tech patterns and boilerplate you reach for often
- Market research summaries
- Regulatory info for agriculture venture
- Anything you'd Google twice — save it here instead
